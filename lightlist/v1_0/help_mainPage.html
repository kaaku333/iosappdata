lightlist について

About lightlist

この App は シンプルで使いやすい Apple Watch 用の買い物リストを提供します。
買い物リストの中であらかじめ買いたい物だけを絞り込んだ チェックリストを Apple Watch で使えます。
This app provides a simple and easy-to-use shopping list for Apple Watch.
You can use a checklist on your Apple Watch that narrows down only the items you want to buy in advance from the shopping list.
↑

使い方概要

Overview of how to use

1. まず iPhone で買い物リストを作ります。
買い物リストは iPhone の本Appで作るか、他のApp(現在はCarnbonFin Outlinerのみサポート)のリストのURLを登録します。
2. 買い物に行く前に iPhone で買い物リストの中の不要な項目をチェックします。
Apple Watch の買い物リストに表示したくない項目は、iPhone で項目にチェックを入れておきます。他の App のリスト場合は、その App で不要項目にチェックします。
3. 買い物に行ったら Apple Watch で買い物リストを使います。
Apple Watch の lightlist App でその店用のリストを選び、買ったものからチェックしていきます。チェックした項目をまとめて削除することもできます。
1. First, create a shopping list on your iPhone.
Create a shopping list with this iPhone app, or register the URL of the list of other apps (currently only CarnbonFin Outliner is supported).
2. Check unnecessary items in the shopping list on your iPhone before going shopping.
Check the items on your iPhone if you don't want them to appear in your Apple Watch shopping list. For a list of other apps, check the unwanted fields in that app.
3. When you go shopping, use the shopping list on your Apple Watch.
Use the lightlist App on your Apple Watch to select a list for that store and check the ones you bought first. You can also delete all the checked items at once.

iPhone App と Apple Watch App の役割

Roles of iPhone App and Apple Watch App

iPhone App の役割
全てのリストを登録、管理する。
全てのリスト項目を追加、編集、管理する。
Apple Watch App に送らない項目をチェックでフィルターする。
更新したリストおよび項目を Apple Watch App に送信する。
Apple Watch App からの要求に応じてリストや項目を送信する。
Apple Watch App の役割
近々必要なリストを管理する。
今使う項目をチェック、編集する。
リストや項目を iPhone App から受け取る。
Apple Watch App でリストや項目を削除、編集しても iPhone には反映されない。
iPhone App Role
Register and manage all lists.
Add, edit and manage all list items.
Check to filter items that are not sent to the Apple Watch App.
Send updated lists and items to the Apple Watch App.
Submit lists and items as requested by the Apple Watch App.
Apple Watch App Role
Manage the lists you will need soon.
Check and edit the items you are using now.
Receive lists and items from the iPhone App.
Deleting or editing a list or item in the Apple Watch App does not take effect on the iPhone.

本 App でリストを作る方法

How to make a list with this app

1. リストの追加
「新規リスト追加」の入力欄をタップして、リストの名前を入れてから「リスト追加」ボタンを押す。「登録ずみリスト」の先頭に登録したリストが表示されます。
2. リストの選択
「登録ずみリスト」から項目を追加・変更したいリスト名を選んでタップすると、項目編集画面に移ります。他AppのリストはこのAppではなく、該当のAppで項目編集します。
3. 項目の追加
+ ボタンの右側をタップすると新しい項目名が入力でき、改行で項目として追加できます。
一括登録の四角をタップするとそこに改行で区切った複数の項目名をまとめて入力できます。「各行を項目として追加」ボタンを押すとまとめて登録されます。
項目の編集
登録された項目名をタップすると項目名を編集することができます。
項目名を左にドラッグすると削除ボタンが出てきて項目を削除することができます。
画面右上の編集ボタンをタップすると、各項目の右側の３本線をドラッグして項目を並び替えることができます。終わったら右上の完了ボタンを押します。
リスト名の変更
左上のリスト名をタップすると、リスト名を変更することもできます。
1. Add list
Tap the "Add new list" input field, enter the name of the list, and then press the "Add list" button. The registered list is displayed at the top of the "Registered list".
2. Select list
Select the list name you want to add / change from the "Registered list" and tap it to move to the item edit screen. For the list of other apps, edit the item with the corresponding app instead of this app.
3. Add item
+ Tap the right side of the button to enter a new item name and add it as an item with a line break.
Tap the large square below of Collective registration to enter multiple item names separated by line breaks. Click the "Add each line as an item" button to register them all at once.
Edit items
You can edit the item name by tapping the registered item name.
If you drag the item name to the left, the delete button will appear and you can delete the item.
By tapping the edit button on the upper right of the screen, you can rearrange the items by dragging the three lines on the right side of each item. When you're done, press the Done button in the upper right.
Rename list
You can also change the list name by tapping the list name on the upper left.

他App のリストを使う方法

How to use the list of other apps

他App として CarbonFin Outliner が使えます。
1. Outliner でリストの作成
CarbonFin Outliner で買い物リストを作ってください。階層があっても構いませんが、lightlist では階層を無くしてフラットなリストになります。
2. Outliner でリストのPublish
lightlist で使いたいリストの設定画面で「Publish」チェックを入れてください。その下の「Copy URL」を押して、リストのURLをコピーしてください。
3. lightlistでリストのURLのペースト
lightlist でリストデータURL登録の入力欄にタップを2回すると、「ペースト」と表示されるので、それをタップして コピーしたURLをペーストしてください。
4. リストの登録
入力欄の下のリスト追加ボタンを押してURLにリストを追加します。うまく登録できれば、リストのタイトルが登録ずみタイトルの先頭に追加されます。リストが追加されない場合は、URLが間違っていないかブラウザで確認してみてください。
CarbonFin Outliner can be used as another app.
1. Create a list with Outliner
Create a shopping list with CarbonFin Outliner. It doesn't matter if there is a hierarchy, but lightlist eliminates the hierarchy and makes the list flat.
2. Publish list with Outliner
Check "Publish" on the setting screen of the list you want to use in lightlist. Press "Copy URL" below it to copy the URLs in the list.
3. Paste the list URL with lightlist
When you tap twice in the input field of list data URL registration in lightlist, "Paste" is displayed. Tap it and paste the copied URL. Please give me.
4. List registration
Click the Add List button below the input field to add the list to the URL. If you can register successfully, the title of the list will be added to the beginning of the registered title. If the list isn't added, check your browser for the wrong URL.

登録ずみリスト一覧の編集方法

How to edit the registered list

リストの削除
リスト名を左にドラッグすると削除ボタンが出てくるのでそれを押してリストを削除できます。
リストの並び替え
画面右上の編集ボタンを押すと、リストの右側の３本線をドラッグすることでリストを並び替えできます。
リスト名の変更
本Appで作ったリストは、リスト項目の編集画面で変更できます。他AppのリストはそのAppでリスト名を変更してからリスト名の右にある丸矢印を押してリスト名を読み込み直してください。
Delete list
Drag the list name to the left and a delete button will appear. You can press it to delete the list.
Sort list
If you press the edit button on the upper right of the screen, you can rearrange the list by dragging the three lines on the right side of the list.
Rename list
The list created by this app can be changed on the edit screen of the list item. For the list of other apps, change the list name in that app and then press the circle arrow to the right of the list name to reload the list name.

Apple Watch のリスト一覧画面

Apple Watch lists screen

コンプリケーションへの登録
ウォッチフェースに lightlist をコンプリケーションとして登録すると、ウォッチフェースから直接 Appを開けます。
リストの登録
iPhoneで登録や編集したリストは Apple Watch にも自動的に同期されます。同期されてなかった時は、リロードボタンを押して全てのリストを同期するか、iPhone で個別にリストを選んで送信します。
リストの選択
リスト名の一覧画面から使いたいリスト名をタップすると項目一覧画面に移ります。
リストの編集
リストの削除は iPhone と同じくリスト名を左ドラッグして、削除ボタンを表示して押します。リストの移動は移動したいリスト名を長押しすると上下に動かせます。リスト名の変更は iPhone で変更して同期してください。
リストの再読み込み
iPhone にあるリストを読み込み直すときはリロードボタンを押してください。
Apple Watch のリストを一旦全クリアする場合はクリアボタンを押してください。
Registration for complications
If you register a lightlist as a complication on your watch face, you can open the app directly from your watch face.
Register list
Lists registered or edited on iPhone are automatically synced to Apple Watch. If it's not synced, press the reload button to sync the entire list, or select the list individually on your iPhone and send it.
Select list
Tap the list name you want to use from the list name list screen to move to the item list screen.
Edit list
To delete the list, drag the list name to the left as in the iPhone to display the delete button and press it. You can move the list up and down by long-pressing the list name you want to move. Please change the list name on your iPhone and synchronize.
Reload list
Press the Reload button to reload from the iPhone list.
To clear the entire list of Apple Watch once, press the clear button.

Apple Watch の項目一覧画面

Apple Watch items screen

画面の表示物
左上にはリスト一覧画面に戻るボタンがあります。また、今日の日付（月はありません）とチェックずみ項目数と全項目数があります。チェックずみ項目数は動的に更新されます。
項目一覧の先頭にはリスト名があります。リスト名の右には項目の保存状態を示すクリップボタンがあり、タップすると状態が切り替わります。初期値の点線表示では項目一覧画面に入るたびに毎回項目をダウンロードし直しますが、実線表示の時は画面を抜けても項目を覚えています。
項目一覧の最後にはチェックした項目を削除するボタンがあります。
項目一覧はこの画面に入ったときに iPhone 経由で最新のデータを転送します。もし更新されない場合は iPhone で該当リストに全部チェックがついていないか確認してください。チェックした項目は表示しません。
項目のチェック
項目名をタップするとチェックされ、もう一度タップするとチェックが消えます。
項目の削除
項目名を左にドラッグして削除ボタンを押します。
チェックした項目の削除
チェックした項目の削除はチェックした項目を削除するボタンを押します。
項目の並べ替え
項目を長押しするとドラッグで移動できます。
Display on the screen
There is a button to return to the lists screen on the upper left. In addition, there are today's date (there is no month), the number of checked items, and the total number of items. The number of checked items is updated dynamically.
There is a list name at the beginning of the item list. To the right of the list name is a clip button that shows the saved state of the item. When you tap it, the state changes, and the dotted line of the initial value re-downloads the item each time you enter the item list screen, but the solid line remembers the item even if you exit the screen.
At the end of the item list, there is a button to delete the checked items.
The item list will transfer the latest data via iPhone when you enter this screen. If it doesn't update, check your iPhone to see if all the lists are checked. Checked items are not displayed.
Check item
Tap the item name to check it, and tap it again to remove the check.
Delete item
Drag the item name to the left and press the delete button.
Delete checked items
To delete the checked items, press the button of delete checked items.
Sort items
Press and hold an item to move it by dragging.
